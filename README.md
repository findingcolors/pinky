## Theme Help

**How to change the background/wave colors**  
##### Light Pink *(top left)*:  
In the CSS, find:

```css
body {  
background-color: #fff8e8;  /* -- change the hex code -- */
```

##### Dark Pink *(bottom right)*:
In the HTML find:

```html
fill:#ffe8e2; <!-- change the hex code -->
```
